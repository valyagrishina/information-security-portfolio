# Исследование мессенджера MAX

## О проекте
Исследование безопасности и функциональных возможностей мессенджера MAX.

## Цель работы
Проведение анализа приложения, изучение аспектов защиты информации и оценка механизмов безопасности.

## Выполненные задачи
- анализ особенностей работы приложения;
- исследование механизмов защиты данных;
- изучение вопросов конфиденциальности;
- подготовка аналитического отчета по результатам исследования.

## Результат
Подготовлен отчет с описанием проведенного исследования, использованных методов и полученных выводов.
