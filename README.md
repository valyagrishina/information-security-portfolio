# Портфолио по цифровой криминалистике

Портфолио проектов по цифровой криминалистике с фокусом на анализ цифровых артефактов, пользовательских следов, а также скрытой и защищённой информации.

## Проекты

- [Исследование мессенджера MAX](./messenger-max) — методические рекомендации для экспертов по поиску и исследованию артефактов мессенджера MAX: анализ пользовательских данных, следов и логов действий пользователя.

- [Поиск скрытой информации](./hidden-information-search) — методические рекомендации по поиску, обнаружению и анализу скрытой и защищённой информации на цифровых носителях.

- [Исследование цифровых следов использования веб-браузеров](./browser-forensics) — методические рекомендации по исследованию артефактов веб-браузеров в рамках компьютерно-технической экспертизы и цифровой криминалистики: анализ истории посещений, пользовательских данных, кэша, файлов cookie и других следов активности.
