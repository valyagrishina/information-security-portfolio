# Исследование мессенджера MAX

## О проекте
Исследование безопасности и функциональных возможностей мессенджера MAX.

## Цель работы
Проведение экспертного исследования артефактов мессенджера Max, в том числе пользовательской информации и следов, изучение аспектов защиты информации.

## Выполненные задачи
- анализ особенностей работы десктопной версии мессенджера Max;
- исследования порядка хранения пользовательской информации;
- анализ пользовательских следов и логов действий пользователя;
- подготовка аналитического отчета по результатам исследования.

## Результат
Подготовлен отчет с описанием проведенного исследования, использованных методов и полученных выводов.
