# Методические рекомендации по поиску скрытой и защищенной информации

## О проекте
Проект посвящен изучению методов поиска, обнаружения и анализа скрытой и защищенной информации на цифровых носителях.

## Цель работы
Разработка методических рекомендаций по выявлению скрытой и защищенной информации с использованием методов экспертного анализа.

## Рассмотренные направления
- поиск скрытых данных;
- анализ цифровых объектов;
- методы обнаружения защищенной информации;
- применение инструментов цифровой криминалистики.

## Результат
Подготовлены методические рекомендации, содержащие описание подходов и методов поиска скрытой и защищенной информации.
