# Исследование цифровых следов использования веб-браузеров

## О проекте

Методические рекомендации, посвященные исследованию цифровых следов использования веб-браузеров в рамках компьютерно-технической экспертизы и цифровой криминалистики.

## Цель работы

Разработка методических рекомендаций по выявлению, извлечению и анализу браузерных артефактов, позволяющих установить факты пользовательской активности и сформировать последовательность действий пользователя при использовании веб-браузера.

## Выполненные задачи

- анализ структуры хранения данных популярных веб-браузеров;
- исследование основных браузерных артефактов (History, Cookies, Login Data, Cache, Bookmarks и др.);
- разработка алгоритма проведения исследования;
- подготовка примера практического исследования;
- формирование перечня типовых вопросов, решаемых при исследовании браузерных артефактов;
- подготовка методических рекомендаций.

## Результат

Подготовлены методические рекомендации, содержащие алгоритм исследования браузерных артефактов, основные термины, пример практического исследования и рекомендации по анализу цифровых следов использования веб-браузеров.
